## Resources

- [YData](https://ydata.ai/) - A platform for synthetic data generation and quality profiling, used to improve training data for machine learning models, especially in imbalanced or sensitive datasets.
- [FastAPI](https://fastapi.tiangolo.com/) - A high-performance Python web framework for building APIs quickly, often used to deploy machine learning models as RESTful endpoints.
- [ArcGIS](https://www.esri.com/en-us/home) - A powerful geographic information system (GIS) platform used for spatial data analysis, mapping, and visualizing geospatial trends in data science projects.
- [GPT2 1.5B Release](https://openai.com/index/gpt-2-1-5b-release/) - A large-scale transformer-based language model developed by OpenAI, used in data science for natural language processing tasks like text generation, summarization, and classification.
- [OpenAI Agents SDK](https://openai.github.io/openai-agents-python/) -A toolkit for building autonomous AI agents that can use tools, APIs, and perform multi-step reasoning, enabling complex workflows in data science automation and decision-making systems.
- [Hugging Face](https://huggingface.co/) - A hub for state-of-the-art machine learning models and datasets, with [/models](https://huggingface.co/models) offering thousands of pre-trained models for NLP, vision, and audio tasks commonly used in data science applications.
- [IBM Granite](https://www.ibm.com/granite) - A family of foundation models developed by IBM for tasks like code generation, NLP, and data analysis, optimized for enterprise-grade AI applications and customizable for domain-specific data science workflows.
- [GPT4](https://openai.com/index/gpt-4-research/) - OpenAI’s multimodal large language model capable of advanced reasoning and understanding, detailed in the technical report and [system card](https://openai.com/index/gpt-4/#system-card), widely used in data science for code generation, data analysis, and intelligent automation.
- [Ultralytics](https://www.ultralytics.com/) - The creators of YOLOv5 and YOLOv8, providing cutting-edge tools and open-source models for real-time object detection and computer vision applications in data science.
- [Canonical](https://canonical.com/) - The company behind Ubuntu, providing open-source infrastructure, cloud, and AI solutions widely used in data science for managing Linux environments, containers, and scalable ML workflows.
- [Ubuntu Packages Search](https://packages.ubuntu.com/) - A searchable index of software packages available in Ubuntu repositories, useful for finding and installing tools, libraries, and dependencies for data science development environments.
- [Google AI Studio](https://aistudio.google.com) - A web-based IDE by Google for experimenting with generative models like Gemini, useful for prototyping AI workflows and testing prompts in data science and LLM applications.
- [Apache Superset](https://superset.apache.org/) - An open-source data exploration and visualization platform for creating interactive dashboards and running SQL-based analytics, often used in data science for BI reporting and data storytelling.
- [pip-audit](https://pypi.org/project/pip-audit/) - A security tool for Python environments that scans installed packages for known vulnerabilities, helping data scientists maintain secure and compliant project dependencies.
- [Pixi](https://pixi.sh/latest/) - A modern package and environment manager by Prefix.dev, designed for reproducible data science workflows with cross-platform support and fast, declarative environment setup using `pixi.toml`.
- [Astral](https://astral.sh/) - The team behind [uv](https://docs.astral.sh/uv/), a blazing-fast Python package manager and installer written in Rust, designed to dramatically speed up dependency resolution and installs; useful in data science for faster environment setup, reproducibility, and CI/CD pipelines.
- [JAX](https://docs.jax.dev/en/latest/) - A high-performance numerical computing library from Google that brings together NumPy-like APIs with automatic differentiation and GPU/TPU acceleration, widely used in data science for machine learning research and gradient-based optimization.
- [Flax](https://flax.readthedocs.io/en/latest/) - A flexible neural network library built on top of JAX, designed for high-performance machine learning research with a focus on composability and scalability in deep learning workflows.
- [Lambda (Lambda Labs)](https://lambda.ai/) - A company offering cloud and on-premise solutions for deep learning, with [Lambda Cloud](https://cloud.lambda.ai) providing affordable, GPU-powered infrastructure for training large-scale AI models in data science.
- [Google Gen AI SDK](https://cloud.google.com/vertex-ai/generative-ai/docs/sdks/overview) - A developer toolkit for accessing Google’s generative AI models (like Gemini) via Python or REST APIs, used in data science to build intelligent applications for text generation, summarization, code, and data insights.
- [Fastfetch](https://github.com/fastfetch-cli/fastfetch) -  blazing‑fast, highly customizable system information tool (a modern alternative to Neofetch), officially hosted on GitHub.
- [Tenacity](https://tenacity.readthedocs.io/en/latest/) - A Python library for adding robust retry logic to code, commonly used in data science to handle flaky APIs, unstable I/O operations, or transient computation failures gracefully.
- [Gensim](https://radimrehurek.com/gensim/) - A Python library for topic modeling and document similarity analysis, widely used in data science for building word embeddings (e.g., Word2Vec) and performing unsupervised NLP tasks on large text corpora.
- [OpenAI o1 + o1-mini](https://openai.com/index/openai-o1-mini-advancing-cost-efficient-reasoning/) - A compact, cost-effective reasoning model in the o1 family optimized for coding and STEM tasks; its safety characteristics are documented alongside the main o1 model in the [system card (Dec 5, 2024)](https://openai.com/index/openai-o1-system-card/)
- [Pytesseract](https://pypi.org/project/pytesseract/) - A Python wrapper for Google’s Tesseract-OCR engine, used in data science to extract text from images for tasks like document processing, image annotation, and OCR-based data extraction.
- [Triton](https://openai.com/index/triton/) - A Python-based programming language and compiler for writing high-performance GPU code with ease, often used in data science and deep learning to optimize custom kernels beyond standard libraries like PyTorch or TensorFlow.
- [Grok 1](https://x.ai/news/grok-os) - A frontier large language model developed by xAI (founded by Elon Musk), designed for reasoning and real-time information tasks, with open weights released for research and data science experimentation.
